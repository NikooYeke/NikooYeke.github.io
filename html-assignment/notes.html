A list of 5 my favorite tags

body tag
Is the main body of the document, All the content of the web page must be added inside
nav tag
The navigation of the page, this will contain the links to other pages of the website
h tag
We can use this tag for the heading of the page. h1 is most important heading and h2, h3,... use for other heads.
strong tag
This tag used for a string of highly emphasized, much more important text
hr tag
This tag represents a thematic break in the content
